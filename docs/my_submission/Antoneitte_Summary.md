## Python
Python is an easy-to-read programming language used to give instructions to computers. It is popular in fields like data science, automation, and web development because it is simple to learn, powerful, and supported by many tools and a large community. 

## Marimo Notebook
Marimo Notebook is an interactive tool for writing and running Python code where code, explanations, and outputs appear in one place. It is reactive, meaning outputs update automatically when the code changes. It is commonly used for data analysis, learning Python, testing ideas, and creating reports

## Git-version-control 
Git is a free, open-source version control system that helps track and manage changes to files during software development. It allows multiple people to work on the same project without overwriting each other’s work and makes it easy to return to previous versions. Git organizes projects into three main areas: the Working Directory (where you edit files), the Staging Area (where you prepare changes), and the Repository (where changes are permanently saved as commits). It is widely used by developers because it protects work, tracks changes, supports collaboration, and is an essential skill in many tech jobs. 

## Plotly-Visualization
Plotly is a versatile library for creating interactive, professional, and web-ready visualizations. It supports over 40 chart types, including 3D plots and maps, and offers features like hover info, zooming, filtering, animations, and easy sharing. It’s suitable for notebooks, dashboards, and publications.

## Polars-data-wrangling
Polars is a modern, high-performance Python DataFrame library written in Rust. It is 5–10× faster than Pandas, memory-efficient, and supports parallel execution and “lazy mode” for large datasets. Polars offers a readable, expressive API with method chaining and a powerful expression language, along with strong typing that catches errors early and improves IDE support. Compared to Pandas, it is faster, uses less memory, provides full parallelism, and enforces type safety, making it ideal for scalable, efficient data workflows.

## Visual Studio Code
VS Code (Visual Studio Code) is a free, cross-platform code editor by Microsoft that highlights syntax, catches errors, runs code, and integrates with Git. It supports many programming languages, includes a built-in terminal, and can be extended with thousands of plugins. 

## UV Package Management
UV is a fast and reliable Python package manager written in Rust. It’s much quicker than pip, automatically handles virtual environments and dependencies, and works seamlessly with pyproject.toml. With UV, you don’t need to manually create or activate venvs, and installing or adding packages is simpler and faster

## Markdown
Markdown is a simple and lightweight way to format text using plain symbols instead of menus or buttons. It lets you make text bold or italic, create lists, add links, insert images, write code, and more, all with easy-to-remember syntax. For example, **bold** becomes bold, *italic* becomes italic, - item creates a bullet list, and [link](https://example.com) becomes a clickable link. Markdown is widely used across coding platforms like GitHub, note-taking apps like Notion, messaging tools like Slack and Discord, blogs, and documentation sites because its files are plain text, lightweight, and easy to share. It’s especially useful for quickly formatting content without a mouse, keeping text clean and professional, and working across multiple apps, making it ideal for writers, programmers, and students.

## Data Science
Data science involves studying and analyzing data to gain useful insights and make better decisions. Data scientists gather and prepare data, analyze it, build models to make predictions, and explain their findings. It is applied in many fields like business, healthcare, science, government, and the environment to solve problems and improve processes.
